## Team Members
  Arielle Aarvig (AarvigA)
## Project Description  
  Training an AI to classify MRI scans of the brain to identify rare neurological diseases. These diseases include:
  - Walker-Warburg Syndrome (WWS)
  - Pachygyria with Cerebellar Hypoplasia
  - Moyamoya Disease
  - Hallervorden-Spatz Disease (PKAN)
  - Fukuyama Muscular Dystrophy
  
Training will be accomplished using an exerpt from the following Kaggle dataset:
  https://www.kaggle.com/datasets/ahsanneural/rare-neurological-diseases-mri-curated-edition

More specifically, the first 70 images of the Train data and the first 15 images of both Val and Test data for each disease classification were used in a folder titled "RareNeuroReduced" to reduce the amount of data used.

In the code, edit the variable "data_path" to reflect the location of "RareNeuroReduced".
